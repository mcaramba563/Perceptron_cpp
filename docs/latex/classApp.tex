\doxysubsection{Detailed Description}
Provides a command-\/line interface for training and using a \doxylink{classPerceptron}{Perceptron} model. 

The \doxylink{classApp}{App} class wraps a default and current \doxylink{classPerceptron}{Perceptron} instance, offering methods to predict, train, save/load models, and manage model configurations.
